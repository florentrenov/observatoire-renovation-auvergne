Ce que dit la note source

La note Obsidian présente le Puy-de-Dôme comme une fiche territoriale hub pour le département. Elle le rattache à l'Auvergne et le relie à Clermont-Ferrand ainsi qu'à Clermont Auvergne Métropole. Elle ne cherche pas encore à produire une monographie départementale exhaustive ; elle organise les sujets à suivre pour l'observatoire.

Les sujets listés donnent une image claire du rôle du département dans le graphe : politiques départementales liées à l'habitat et à l'énergie, programmes locaux de rénovation, acteurs institutionnels et associatifs, projets exemplaires, évolution du DPE dans le parc résidentiel, besoins en audit énergétique pour les maisons individuelles anciennes, parcours de rénovation globale, mobilisation de MaPrimeRénov', financement par les CEE, accompagnement avec France Rénov' et Mon Accompagnateur Rénov', chauffage au bois, granulés et performance réelle des logements.

Rôle dans le graphe

Le Puy-de-Dôme est le noeud territorial central de l'observatoire. Il permet de ne pas présenter la rénovation énergétique comme une suite de fiches techniques abstraites. Chaque sujet prend sens parce qu'il est situé : un DPE est produit sur un logement, un audit décrit un bâtiment, une aide est mobilisée par un ménage, un accompagnateur intervient dans un contexte local, une rénovation globale dépend d'un parc bâti et d'entreprises disponibles.

Le territoire agit donc comme une charpente éditoriale. Il relie les fiches documentaires, les acteurs, les aides, les veilles et les études de cas. Cette fonction est particulièrement importante pour éviter deux pièges : un site trop technique, qui parlerait seulement de dispositifs, et un site trop narratif, qui raconterait des cas sans les relier aux outils d'analyse.

Lecture territoriale

La note source met en avant plusieurs questions : quels types de logements sont les plus concernés par les besoins de rénovation énergétique, quels freins apparaissent pour les propriétaires de maisons individuelles, quels acteurs locaux accompagnent le passage du diagnostic aux travaux, et quelles données publiques peuvent nourrir l'observatoire. Ces questions doivent structurer la suite du site.

Le Puy-de-Dôme combine des situations urbaines, périurbaines et rurales. Cette diversité justifie une lecture par échelles : département, métropole, commune, puis cas concret. Clermont-Ferrand permet d'entrer dans les enjeux urbains ; Clermont Auvergne Métropole sert d'échelle de coordination ; la maison des années 70 montre comment les fiches techniques deviennent un parcours réel.

Sections manquantes à documenter

La note ne contient pas encore de chiffres locaux détaillés sur le parc de logements, les classes DPE, les rénovations engagées, les acteurs de terrain ou les aides locales. Elle signale un besoin de données publiques. Pour transformer cette page en ressource de référence complète, il faudra ajouter des sources territoriales vérifiées, distinguer les échelles communale, métropolitaine et départementale, puis construire des indicateurs suivis dans le temps.
